# OSINT — Index

Osservazioni estratte da POE-OSINT e note investigative.

## Come usare
- Crea una nota per ogni soggetto/caso di interesse
- Linka le entità estratte (IP, domini, email)
- Usa i tag `#ip`, `#email`, `#domain`, `#cve`

## Template
```
# Soggetto: [nome]

**Data:** 
**Score:**
**Entità principali:**
- 

**Note:**

**Correlazioni:** [[...]]
```
